# لوحة التحكم - Admin Dashboard

## معلومات الدخول

- **رابط الوصول:** `https://haithamsalama.com/admin.html`
- **اسم المستخدم:** `admin`
- **كلمة السر:** `Lara@2014`

## الميزات

### 1. إدارة الملفات (Files Management)

- **إضافة ملف جديد:**
  - اكتب اسم الملف
  - أدخل رابط التحميل
  - أضف وصف اختياري
  - اضغط "إضافة ملف"

- **تعديل ملف:**
  - اضغط زر "تعديل" بجانب الملف
  - عدّل البيانات المطلوبة
  - اضغط "تحديث الملف"

- **حذف ملف:**
  - اضغط زر "حذف" بجانب الملف
  - أكد عملية الحذف

### 2. إدارة البرامج (Programs Management)

- **إضافة برنامج جديد:**
  - اسم البرنامج بالعربية والإنجليزية
  - وصف البرنامج
  - رابط الفيديو (YouTube)
  - رابط التحميل
  - صورة البرنامج (يتم ضغطها تلقائياً)
  - اضغط "إضافة برنامج"

- **تعديل برنامج:**
  - اضغط زر "تعديل" بجانب البرنامج
  - عدّل البيانات المطلوبة
  - اضغط "تحديث البرنامج"

- **حذف برنامج:**
  - اضغط زر "حذف" بجانب البرنامج
  - أكد عملية الحذف

## نوع قاعدة البيانات

- **الطريقة الحالية:** localStorage (تخزين محلي على المتصفح)
- **حدود التخزين:** حوالي 5-10 MB حسب المتصفح
- **الصور:** يتم تحويلها إلى Base64 وحفظها مباشرة

## الانتقالات إلى الصفحات الفعلية

- **صفحة الملفات:** `/files.html` - تحمل البيانات من localStorage تلقائياً
- **صفحة البرامج:** `/downloads.html` - تحمل البيانات من localStorage تلقائياً

## ملفات مهمة

- `admin.html` - واجهة لوحة التحكم
- `admin.js` - منطق لوحة التحكم والبيانات
- `admin-helper.js` - دوال مساعدة لتحديث الصفحات الفعلية

## خطوات الاستخدام

1. افتح `/admin.html`
2. أدخل بيانات الدخول (admin / Lara@2014)
3. استخدم التبويبات للتبديل بين الملفات والبرامج
4. أضف أو عدّل أو احذف كما تشاء
5. ستظهر التغييرات تلقائياً على الصفحات الفعلية

## ملاحظات أمان

- ⚠️ بيانات الدخول مخزنة في الكود (يفضل نقلها لخادم)
- ⚠️ localStorage يمكن الوصول إليه عبر أي صفحة
- ⚠️ للأمان الحقيقي، استخدم قاعدة بيانات سحابية آمنة

## التوسع المستقبلي

- نقل قاعدة البيانات إلى Firebase
- إضافة نظام النسخ الاحتياطي
- تشفير كلمة المرور
- إضافة سجل العمليات (Audit Log)
